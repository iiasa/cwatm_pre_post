Calibration
-----------


Originally from Lisflood
https://github.com/ec-jrc/lisflood-calibration
by Hylke Beck and Peter burek (2014), modified by Feyera Aga Hirpa,Valerio Lorini and Alfieri Lorenzo.
The submodule Hydrostats was created 2011 by Sat Kumar Tomer (modified by Hylke Beck).

Modified 2021 for CWatM by peter Burek

Software required

    pcraster > 0.41 http://pcraster.geo.uu.nl/
    Python 3.8
    postscript (for figures, you can avoid it though)

Python packages

    netCDF4 https://pypi.org/project/netCDF4/
    pandas (pip install ...)
    numpy (pip install ...)
    matplotlib (pip install ...)
    deap (https://deap.readthedocs.io/en/master/installation.html)
